# ?? Project Guardian: Terminal Management System

## ?? Overview
A full-stack management solution designed to track transport trips and manage driver assignments. This project was built to demonstrate proficiency in relational databases, RESTful APIs, and responsive frontend design.

## ?? Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Navy & Gold Theme), JavaScript (Fetch API)
- **Backend:** Node.js, Express.js
- **Database:** SQLite3 (Relational)

## ? Key Features
- **Real-time CRUD:** Create, Read, and Delete trips and drivers.
- **Relational Mapping:** Assign specific drivers to trips using Foreign Keys.
- **Live Search:** Instant filtering of trip data by destination, status, or driver.
- **Reporting:** Print-ready CSS for generating formal daily trip reports.
